# SIRDATA

**SIRDATA** es una herramienta desarrollada para el Sistema Estadístico
Municipal (SEM) de la Alcaldía de Pereira que permite analizar y
diagnosticar bases de datos provenientes de registros administrativos.

La aplicación facilita el aprovechamiento estadístico de la información
mediante el análisis automático de la estructura de los datos, la
generación de diccionarios de datos y la evaluación de indicadores
básicos de calidad.

## Funcionalidades

-   Análisis estructural de bases de datos
-   Generación automática de diccionario de datos
-   Evaluación de indicadores de calidad de la información
-   Identificación de variables y tipos de datos
-   Exportación del diccionario de datos en formato Excel

## Formatos soportados

La herramienta permite analizar bases de datos en los siguientes
formatos:

-   Excel (.xlsx)
-   CSV (.csv)
-   JSON (.json)

## Tecnologías utilizadas

-   Python
-   Streamlit
-   Pandas
-   NumPy

## Propósito

SIRDATA busca fortalecer el uso estadístico de los registros
administrativos y apoyar los procesos de gestión de datos en el Sistema
Estadístico Municipal.

## Desarrollo

Herramienta desarrollada en el marco del fortalecimiento del Sistema
Estadístico Municipal de la Alcaldía de Pereira.

## Versión

SIRDATA 1.0
